workerman-todpole
=================

蝌蚪游泳交互程序-HTLM5+WebSocket+Workerman , rumpetroll server rewrite using php.


在自己的服务器上安装部署
==================

1、下载或者clone代码到本地

2、进入目录运行./bin/workermand start

3、浏览器访问地址  http://ip:8383 如图：

![小蝌蚪游戏截图](todpole-browser.png)
